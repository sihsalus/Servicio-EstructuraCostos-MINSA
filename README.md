# Servicio de Costo Estandar de CPMS

Este proyecto es una API en Node.js (Express) construida con TypeScript que genera la estimación de costos estandar para los procedimientos médicos o procedimientos sanitarios en las **Instituciones Prestadoras de Servicios de Salud** . Está diseñado para integrarse parcialmente con OpenMRS.

---

## Stack
 * Express
 * Typescript
 * Sequelize
 * Axios / cheerio
 * pnpm
